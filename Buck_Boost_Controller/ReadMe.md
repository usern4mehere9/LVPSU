-Low Voltage Power Supply Unit of Formula Student team "Black Hornets" from the University of Novi Sad Faculty of Technical Sciences 

-This folder is to be used for storing any and all documentation and information, as well as schematics for the Buck Boost controller that will be used in the LV Regulator

-The Folder Documentation is to be used for storing any and all documentation regarding components used in the schematic, and or any documentation regarding test boards & usecase documentation given by component manufacturers

-The Folder Buck_Boost_Controller is a KiCad Project file, it also has the local libraries for the components used and imported from the manufacturers saved locally in it.

-The inside of this ReadMe.md file is to be used for documentation of changes made by any user making edits in this project, as well as explanation of every change made and the reason why it was made, and how things work inside the project.
